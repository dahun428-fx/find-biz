**헤더 (Header)**
- 로고 (클릭 시 홈으로)
- 탭 네비게이션: 전체 SKILLS / 내 SKILLS / 공유받은
- 사용자 메뉴: @username 드롭다운 (로그아웃)

**서브 헤더 (Sub Header)**
- 사이드바 토글 버튼 (☰)
- 검색바 (실시간 검색)
- 정렬 드롭다운 (최신순/오래된순/이름순/즐겨찾기순)
- 새 SKILL 등록 버튼 (primary)

**사이드바 (Sidebar)**
- 접기/펼치기 가능
- 카테고리 트리 구조
- 계층별 들여쓰기
- 클릭 시 필터링 적용

**메인 콘텐츠 (Main Content)**
- 카드 그리드 (4열, 반응형)
- 페이지네이션 (하단)

#### 2.2 SKILL 카드 컴포넌트
```
┌─────────────────────┐
│ Frontend Design     │ ← 제목
│                     │
│ A comprehensive     │ ← 설명 (2-3줄 요약)
│ guide for modern    │
│ UI/UX design...     │
│                     │
│ 개발 > 프론트엔드    │ ← 카테고리
│                     │
│ ⭐ 5      [♡ 즐겨찾기]│ ← 즐겨찾기 수 + 버튼
└─────────────────────┘
```

**카드 요소:**
- 제목 (title, 1줄, 말줄임)
- 설명 (description, 2-3줄, 말줄임)
- 카테고리 경로 (최대 3depth 표시)
- 즐겨찾기 수
- 즐겨찾기 버튼 (하트 아이콘, 토글)

**상호작용:**
- 카드 클릭 → 상세 페이지 이동
- 즐겨찾기 버튼 클릭 → 즐겨찾기 추가/제거 (즉시 반영)
- 카테고리 클릭 → 해당 카테고리로 필터링

**반응형:**
- 데스크톱: 4열
- 태블릿: 2-3열
- 모바일: 1열

```
│  └─────────────────────────────────────────────────────┘   │
│                                                             │

│  ───────────────────────────────────────────────────────   │
│  [#design] [#ui] [#frontend] [#ux]                          │ ← 클릭 시 필터링
│                                                             │
│  GitHub                                                     │
│  ───────────────────────────────────────────────────────   │
│  🔗 https://github.com/anthropics/claude-code/...           │

│  개인 메모 (본인만 표시)                                    │
│  ───────────────────────────────────────────────────────   │

│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**액션 버튼:**
- 즐겨찾기: 하트 아이콘 + 개수 표시

- 공유: 모달 열기 (이메일 입력)
- 수정: 본인 SKILL만 표시
- 삭제: 본인 SKILL만 표시

**권한별 표시:**
- 본인 SKILL: 모든 버튼 표시
- 다른 사용자 SKILL: 즐겨찾기, 복사, 공유만 표시
- 공유받은 SKILL: 모든 버튼 표시 (독립적으로 관리)

## 디자인 토큰 (shadcn/ui 기반)

### 색상
- **Primary**: Blue (SKILL 등록, 주요 액션)
- **Secondary**: Gray (보조 버튼)
- **Destructive**: Red (삭제)
- **Muted**: Light Gray (배경, 비활성)
- **Accent**: Purple (AI 추천 표시)

### 타이포그래피
- **제목 (H1)**: 2rem, font-bold
- **부제목 (H2)**: 1.5rem, font-semibold
- **본문**: 1rem, font-normal
- **작은 텍스트**: 0.875rem, font-normal

### 간격
- **카드 간격**: 1rem (gap-4)
- **섹션 간격**: 2rem (gap-8)
- **패딩**: 1rem (p-4)

### 반응형 브레이크포인트
- **모바일**: < 640px (1열)
- **태블릿**: 640px ~ 1024px (2-3열)
- **데스크톱**: > 1024px (4열)

## 인터랙션

### 애니메이션
- 카드 호버: scale(1.02), shadow 증가
- 버튼 클릭: scale(0.98)
- 모달 열기: fade-in + scale-up
- 페이지 전환: fade

### 피드백
- 로딩 상태: 스피너 + 텍스트
- 성공: 토스트 알림 (녹색)
- 에러: 토스트 알림 (빨간색)
- 즐겨찾기 추가: 하트 애니메이션

### 접근성
- 키보드 네비게이션 지원
- ARIA 레이블 추가
- 포커스 표시 (outline)
- 색상 대비 WCAG AA 준수
