# 관리자 대시보드 UI 디자인

## 관리자 대시보드 페이지 (`/admin`)

## 디자인 가이드

### 색상
- **Admin Badge**: Orange (관리자 표시)
- **Warning Actions**: Red (삭제, 정지)
- **Success Actions**: Green (활성화, 저장)

### 아이콘
- 사용자 관리: 👥
- SKILL 관리: 📚
- 통계: 📊
- 설정: ⚙️

### 권한 표시
- 관리자 페이지 상단에 "관리자 모드" 배지 표시
- 일반 사용자가 접근 시 403 에러 페이지

---

## 접근성
- 관리자 전용 페이지는 명확한 표시
- 위험한 액션(삭제, 정지)은 확인 모달 필수
- 키보드 네비게이션 지원
